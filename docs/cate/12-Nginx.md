## linux命令

```
打开文件夹
cd /
cd name

查看目录
ls

创建文件夹
mkdir name

删除文件夹
rm -rf name
 
-r 就是向下递归，不管有多少级目录，一并删除
-f 就是直接强行删除，不作任何提示的意思
```
