# webui开发流程



## 项目架构

项目使用vue3+ts+vite+pnpm



### 目录

```ts
├──application                    # php代码
├──images                      	  # 图片
├──mock                      	  # mock代码
├──portal                      	  # portal代码
├──src                      	  # vue3代码
│  ├──apis                        # 接口
│  ├──assets                      # 静态资源
│  ├──components                  # 业务公共组件
│  ├──constants                   # 常量
│  ├──design                      # 全局样式
│  ├──enums                       # 公共枚举值
│  ├──ext2vue                     # vue渲染ext组件
│  ├──hooks                       # 公共的hooks
├──WLAN                      	  # 原ext代码


```
